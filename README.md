bapudi
======

What is 'bapudi'?
-----------------
'બાપુડી' is slang word in Gujarati and book means ચોપડી in Gujarati. Together it will be: બાપુડી.

This project has deep inspiration from following projects:

* http://www.urbandictionary.com/
* http://samosapedia.com/
